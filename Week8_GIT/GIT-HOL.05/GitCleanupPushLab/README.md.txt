# Git Cleanup and Push to Remote Lab

## Hands-on ID: Git-T03-HOL_002

### Objective
- Learn how to clean up and push changes to a remote Git repository.

### Estimated Time
10 minutes

### Steps Performed
1. Verified that the master branch is clean.
2. Listed all branches using `git branch -a`.
3. Pulled the remote repository to the master.
4. Pushed local changes to remote.
5. Confirmed that changes are reflected in the GitHub repository.
