# Git Branching and Merging Lab

## Objectives
- Create a new branch, make changes, and merge it back into master.
- Use P4Merge to visualize differences between branches.

## Estimated Time
30 minutes

### Steps Performed
1. Created a new branch (`GitNewBranch`).
2. Added changes and committed them to the branch.
3. Merged the changes into master.
4. Used `git log` and `git difftool` (P4Merge) to visualize and confirm the merge.
